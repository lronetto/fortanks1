Modelo para Importação de Planos de Conta
Instruções:

A planilha para importação deve seguir o formato abaixo:

  • A primeira linha contém os cabeçalhos das colunas.
  • Cada linha seguinte representa um plano de conta.
  • As colunas marcadas com (*) são obrigatórias.
Número (*) Índice (*) Descrição (*) Sequência (*)
Exemplo: 1.1.01 Exemplo: A.1.01 Exemplo: Receitas Operacionais Exemplo: 1
Observações Importantes:
  1. Número e Índice devem seguir um padrão consistente para facilitar a organização.
  2. Sequência deve ser um número inteiro que define a ordem de apresentação do plano de conta.
  3. Evite usar o mesmo número ou índice para planos de conta diferentes.
  4. A importação pode substituir planos de conta existentes se a opção for selecionada.
