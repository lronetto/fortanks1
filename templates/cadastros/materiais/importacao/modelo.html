Instruções:

A planilha para importação deve seguir o formato abaixo:

  • A primeira linha contém os cabeçalhos das colunas.
  • Cada linha seguinte representa um material.
  • As colunas marcadas com (*) são obrigatórias.
Nome (*) Tipo (*) Unidade (*) Valor Unitário (*) PC Código ERP
Exemplo: Parafuso Allen M10 Exemplo: Fixação Exemplo: UN Exemplo: 2.50 Exemplo: PC001 Exemplo: ERP001
Observações Importantes:
  1. Valores unitários devem usar ponto (.) como separador decimal, não use vírgula.
  2. Os campos PC e Código ERP são opcionais, mas recomendados para rastreabilidade.
  3. Tipos válidos incluem: Fixação, Elétrico, Mecânico, Hidráulico, etc.
  4. Unidades válidas incluem: UN, KG, M, M², M³, L, etc.
