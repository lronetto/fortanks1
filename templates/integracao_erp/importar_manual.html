Importação Manual de Arquivo XLS
Voltar

Instruções:

  • Selecione um arquivo XLS exportado do sistema ERP
  • O arquivo deve estar no formato HTML, conforme gerado pelo sistema
  • O sistema irá processar as transações e importá-las para o banco de dados
  • Após a importação, você poderá visualizar as transações na seção "Transações"
Selecione o arquivo .xls gerado pelo ERP
Exemplo de Arquivo

O arquivo XLS deve conter uma estrutura similar a esta:

<table>
  <tr>
    <td>C.Custo: 1234-56 DEPARTAMENTO FINANCEIRO</td>
    <td></td>
    <td></td>
    ...
  </tr>
  <tr>
    <td></td>
    <td>101</td> <!-- Categoria -->
    <td>20/03/2025</td> <!-- Data -->
    <td>NF-12345</td> <!-- Documento -->
    <td>FORNECEDOR XYZ</td> <!-- Emitente -->
    <td></td>
    <td>PAGAMENTO REFERENTE A SERVIÇOS</td> <!-- Histórico -->
    ...
    <td>1.000,00</td> <!-- Valor -->
  </tr>
  ...
</table>
